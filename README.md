# HFSS / Electronics Desktop Project Files

This repository contains ANSYS Electronics Desktop archive files (`.aedtz`).

## Files
- `microsrip antenna @2.4Ghz.aedtz`
- `microstrip of 50 ohm.aedtz`

## Requirements
- ANSYS Electronics Desktop (AEDT)
- HFSS installed

## How to open the files in EDT

### Method 1: Open directly
1. Open **ANSYS Electronics Desktop**.
2. Go to **File > Open**.
3. Browse to the required `.aedtz` file.
4. Select the file and open it.

AEDT will extract and load the project automatically.

---

### Method 2: Extract first, then open
1. In **ANSYS Electronics Desktop**, go to **File > Restore Archive**.
2. Select the `.aedtz` file.
3. Choose a location to restore the project.
4. AEDT will create the project file, usually with extension `.aedt`.
5. Open the restored project in Electronics Desktop.

---

## Notes
- Keep the restored project folder and generated result files together.
- If the project opens in the wrong design environment, open it from **ANSYS Electronics Desktop** instead of a product-specific shortcut.
- These archive files are meant to be opened inside AEDT, not extracted with normal ZIP software.

## Sharing
You can clone or download this repository and then open the `.aedtz` files from AEDT.
